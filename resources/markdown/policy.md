# Politica de Privacidad

Fecha de vigencia: 2026-02-05

## Datos que recopilamos

- Datos de cuenta: nombre, correo y contrasena (cifrada).
- Datos de inventario: productos, precios, cantidades e imagenes.
- Datos tecnicos: registros basicos para seguridad y soporte.

## Uso de la informacion

Usamos los datos para operar la aplicacion, brindar soporte, aplicar seguridad y mejorar la experiencia.

## Compartir datos

No vendemos datos personales. Podemos compartirlos con proveedores tecnicos que ayudan a operar la aplicacion (por ejemplo, hosting y correo), bajo acuerdos de confidencialidad.

## Retencion

Conservamos los datos mientras la cuenta este activa o cuando sea necesario para cumplir obligaciones legales.

## Seguridad

Aplicamos medidas razonables para proteger la informacion, pero ningun sistema es 100% seguro.

## Tus derechos

Puedes solicitar actualizacion o eliminacion de tu informacion cuando corresponda.

## Contacto

Para temas de privacidad, contacta al administrador del sistema.
